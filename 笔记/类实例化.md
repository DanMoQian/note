# 类实例化

> 类加载的时机

1. 实例化
2. 反射
3. 静态字段
4. 静态方法

> 类的实例化顺序

1. 执行静态块
2. 执行构造代码块
3. 执行构造器

注意：

	- 类的实例成员，等价于构造代码块
	- 类的静态成员，等价于静态代码块

> 构造代码块就相当于在构造器中的前面，只有new 构造器才会执行构造代码块和构造器

静态的本质就是类一加载就有空间
